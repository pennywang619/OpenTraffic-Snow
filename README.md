# OpenTraffic-Snow

Public delivery repository for traffic-scene snow detection.

## Delivery Contents

This repository includes:

1. shared library delivery in `artifacts/`
2. example outputs in `examples/`
3. validation summaries in `validation/`
4. public manifest in `manifests/`
5. checksum file in `checksums.txt`

## Quick Start

1. Verify the delivered files with `checksums.txt`.
2. Review `manifests/public_manifest.json`.
3. Review `validation/public_validation_summary.json`.
4. Integrate `artifacts/libsnow_detection.so` into the target Linux runtime.
5. Compare local outputs with the example files in `examples/`.

See `USAGE.md` for the delivery usage steps.
